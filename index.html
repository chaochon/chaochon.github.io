#贪吃蛇

##两个问题

  1. 如何实时的从键盘中读入控制按键，并且蛇及时地响应？
  2. 如何让蛇动起来，形成动画？

##用例
主要参与者：一个玩家
关注点：希望能够记录成绩，并列在排行榜上
主成功场景：

  1. 玩家输入ID，游戏进行初始化
  2. 玩家启动游戏，进行游戏活动
  3. 游戏结束，显示玩家成绩的排名
    ##对象
    主要有四个对象，蛇，苹果，墙，和界面中数据。
  4. 蛇类具有两个数据成员，一个是存储蛇身坐标的链表body，一个是蛇运动方向dir
    另外主要的操纵有移动move，吃苹果tryeat，改变运动方向changDirect,
    判断是否撞墙hitwall和是否是否吃到自己eatItself的操作。
  5. 苹果类具有一个坐标（x,y）数据成员，具有一个随机产生一个新苹果newApple操作
  6. 墙类具有两个点数据成员，用来限定一个矩形的蛇运动范围

  7. ui类具有一个游戏难度等级和分数的数据成员，还有一个自增分数和游戏难度的操作inc。
    类图

    ##主程序框架
    main函数来说明

    int main(){

    initGame();//初始化游戏
do{
    keybordControl();//按键控制
    handlelogic();//处理游戏逻辑
    draw();//绘图
    sleep(10);//控制帧率
 }while(isGameRuning);
return 0;

    }
    keybordControl( )的功能：接受键盘按键输入，主要利用VS自带的GetAsyncKeyState()函数来实现。
    handlelogic( )功能：处理游戏运行逻辑，比如说何时结束游戏，如何加速等主要工作。
    draw( )功能：在处理完游戏逻辑后，将每个对象的图形绘制在窗口中。
    sleep(10) 功能：控制图形的刷新帧率。如果不添加一个计算间隔时间，动画将会一闪一闪的。

主程序的运行流程是：isGameRuning是一个初始化为1的全局变量，用来控制整个程序运行与否。
首先判断isGameRuning是否为真，如果是就进入循环，接受按键，处理游戏逻辑，绘制图形形成
图像。刷新一幅一幅的图形就成了动画，这就解决了开头的问题。

##游戏逻辑
思路：

  1. 首先判断游戏是否结束，如果是这return退出函数
  2. 判断蛇的下一个位置是否咬到自己或者撞墙，如果是则全局变量GameOver置为真

  3. 判断蛇的下一个位置是否吃到苹果，如果是则将苹果头插法加入蛇不移动，随机生成一个新苹果
    并增加成绩（成绩达到条件就蛇加速），否则就蛇移动一步。
    代码实现如下

    void handleLogic(){

    if (GameOver)
    return;
/*判断下一位置是否吃到自己或者撞墙*/
if (snake.eatItself() || snake.hitwall(wall)){
    GameOver = true;
}
if (snake.tryeat(apple)){
    apple.newApple();
    ui.inc();
}
else
    snake.move();

    }

##蛇移动的实现
蛇的移动如下图所示

思路：

  1. 头节点依照自身的方向移动一格

  2. 接下来只需要将后一个节点移动到前一个节点的位置
    需要注意的是，在每个节点移动以前要用pre变量将前一个节点位置保持下来
    代码如下：

    void Snake::move(){

    int xofset[4] = { 0, 0, -1, 1 };
int yofset[4] = { -1, 1, 0, 0 };
/*控制游戏速度*/
int intervalTime = 100 - Speed;
static int lastMoveTick = 0;
int curTicktime = GetTickCount();
if (curTicktime - lastMoveTick < intervalTime){
    return;
}
lastMoveTick = curTicktime;
/*控制游戏速度end*/
LinkList p = body;
int prex = p->next->data.x;
int prey = p->next->data.y;

p->next->data.x += xofset[dir];
p->next->data.y += yofset[dir];
p = p->next;
do{
    int curx = p->next->data.x;
    int cury = p->next->data.y;
    p->next->data.x = prex;
    p->next->data.y = prey;
    prex = curx;
    prey = cury;
    p = p->next;
} while (p->next != NULL);

    }
    注意在EaxyX图形库中，默认坐标系是，X轴以横向向右为正，Y轴纵向向下为正。

##判定碰撞、吃自身和吃苹果的实现
这三个操作的实现大同小异所以写一起
撞墙思路：

  1. 蛇头依据自身方向算出下一个节点坐标
  2. 判断该节点坐标是否在围墙范围之内，如是则返回假，否则返回真

代码实现

bool Snake::hitwall(Wall wall){
    int xofset[4] = { 0, 0, -1, 1 };
    int yofset[4] = { -1, 1, 0, 0 };
    Point prePoint;
    prePoint.x = body->next->data.x;
    prePoint.y = body->next->data.y;
    prePoint.x += xofset[dir];
    prePoint.y += yofset[dir];
    if (prePoint.x < wall.a.x || prePoint.x > wall.b.x-1 || prePoint.y < wall.a.y ||     prePoint.y > wall.b.y-1)
        return true;
    return false;
}

吃自身思路：

  1. 蛇头依据自身方向算出下一个节点坐标
  2. 让蛇身链表与该节点逐个比较，如果发现相同的立马返回真。

代码实现

bool Snake::eatItself(){
int xofset[4] = { 0, 0, -1, 1 };
int yofset[4] = { -1, 1, 0, 0 };
Point prePoint;
prePoint.x = body->next->data.x;
prePoint.y = body->next->data.y;
prePoint.x += xofset[dir];
prePoint.y += yofset[dir];
LinkList p = body;
do{
    if (p->next->data== prePoint)
        return true;
    p = p->next;
} while (p->next != NULL);
return false;
}

吃苹果思路：

  1. 蛇头依据自身方向算出下一个节点坐标
  2. 让该节点与苹果坐标比较，如果相同则返回真，否则返回假

代码实现

bool Snake::tryeat(Apple apple){
    int xofset[4] = { 0, 0, -1, 1 };
    int yofset[4] = { -1, 1, 0, 0 };
    Point prePoint;
    prePoint.x = body->next->data.x;
    prePoint.y = body->next->data.y;
    prePoint.x += xofset[dir];
    prePoint.y += yofset[dir];
    if (prePoint.x == apple.x && prePoint.y == apple.y){
        Point apple1 = { apple.x, apple.y };
        ListInsert(body, 1, apple1);
        return true;
    }
    return false;
    }

##改变蛇方向的实现
思路：

  1. 依据接收到的按键指令分成两类 上下和左右
  2. 如果是上下，则判断按键指令是否为左右，如是则赋值改变蛇方向，否则不变方向

  3. 如果是左右，则判断按键指令是否为上下，如是则赋值改变蛇方向，否则不变方向
    代码如下

    void Snake::changeDirect(Direct direc){

    switch (dir)
{
case up:
case down:
    if (direc == left || direc == right)
        dir = direc;
    break;
case left:
case right:
    if (direc == up || direc == down)
        dir = direc;
    break;
default:
    break;
}

    }
