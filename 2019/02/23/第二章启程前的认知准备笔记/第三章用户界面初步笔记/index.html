1.图像的载入、显示和输出1.图像的载入、显示和输出# 1.图像的载入、显示和输出
Mat类是用于保存图像以及其他矩阵数据的数据结构，默认结构尺寸为0，也可以指定尺寸

1.1 图像的载入 imread()函数

其函数原型
Mat imread(const string& filename, int flag=1)

  • flag = -1新版本中废纸
  • flag = 0 以灰度图像返回
  • flag = 1 转换至彩色图像返回
  • flag = 2 如载入图像的深度16位或者32位，就以相应深度的图像返回，否则以深度位8位 返回
  • flag = 4
    如果flag不在枚举体中，则
  • flag>0 以彩色图像返回
  • flag=0 以灰度图像返回
  • flag<0

图像深度是指存储每个像素所用的位数，也用于量度图像的色彩分辨率。

1.2 图像的显示:imshow()函数

函数原型：
void imshow(const string& winname, InputArray mat)

  • 第一个参数：填写要显示窗口名字
  • 第二个参数：InputArray型待显示的图像

    1.3 创建窗口：namedWindow()函数

    函数原型：
    void namedWindow(const string& winname, int flag = WINDOW_AUTOSIZE)
  • 第一参数：填写窗口名称
  • 第二参数：窗口标识，取CV_WINDOW_NORMAL时，用户可以手动改变窗口大小；取CV_WINDOW_AUTOSIZE, 窗口自动适应图像，且不能手动改变窗口大小

    1.4输出图像：imwrite()函数

    函数原型
    bool imwrite(const string& filename,InputArray img,const vector<int>& params=vector<int>())
  • 第一个参数表示输出的文件名
  • 第二个参数：图像数据
  • 第三个参数：表示特定格式保存的参数编码

阿尔法通道是一个8位的灰度通道，该通道用256级灰度来记录图像中的透明度信息，定义透明、不透明和半透明区域，其中白表示不透明，黑表示透明，灰表示半透明。

  1. 取出感兴趣的区域
    imageROI=image(Rect(x0,y0,L,W))
    其中image为Mat型变量，（x0,y0)为矩形左上角，L,W分别为长和宽
    imageROI=image(Range(x1,x2),Range(y1,y2))
    其中Range(x1,x2)表示列范围，Range(y1,y2)表示行范围
    注意取得的区域图像imageROI与原图像共享内存，对imageROI的操作会反映到image图像上
  2. 融合图像
    addWeighted(image1,a,image2,b,c,printedImage)
    作用是将image1和image2 分别按a,b系数融合再输出到printedImage图像

总结：
创建滑动条函数原型：
int createTrackbar(const string& const string& winname, int *value, int count, TrackbarCallback onChange = 0, void* userdata = 0)

  • 第三个参数value 指的是滑块位置，第四个参数count指的最大滑块位置
  • TrackbarCallback类型的onChange指的是 一个指向回调函数的指针 （每次移动滑块 就会调用该函数）所以 自然地需要回调函数配合使用
  • 另外还有一个配合使用的函数getTrackbarPos(),用于获取当前位置
